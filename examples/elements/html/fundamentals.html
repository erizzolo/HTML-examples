Elementi fondamentali della struttura di un documento

html
l'elemento radice di un documento
con l'attributo lang si specifica la lingua del documento
contiene:
  • head
  • body
head
rappresenta le informazioni sul documento
contiene:
  • title
  • eventuali meta (charset, ...)
  • eventuali link (per risorse aggiuntive, ad es. CSS, Javascript ...)
body
rappresenta l'intero contenuto del documento
