Uniform Resouce Locator

Uno Uniform Resource Locator (abbreviato URL) è un tipo di Uniform Resource Identifier (abbreviato URI), di cui segue la sintassi, usato per localizzare e recuperare una risorsa.

Parti di un URI

Un URI (e quindi un URL) comprende:
  • uno schema seguito da : che spiega come accedere alla risorsa Tipicamente indica il protocollo, come ad esempio http, https, ftp, mailto ...
  • una autorità preceduta da // (opzionale) Essa è formata da:
    • una informazione utente seguita da @ (opzionale) Le credenziali di accesso, ad esempio username oppure username:password
    • un nome host Ad esempio il nome di dominio registrato o un indirizzo di rete
    • un numero di porta preceduto da : (opzionale) Qualora il server usi un numero di porta non standard
  • un percorso formato da segmenti separati da / Simile al percorso di un file all'interno del filesystem.
  • una query preceduta da ? (opzionale) Di solito una sequenza di coppie attributo=valore separate da &
  • un frammento preceduto da # (opzionale) Indica una parte specifica della risorsa, ad esempio un elemento in una pagina html

Esempio

https://translate.google.it/?op=translate&sl=en&tl=it&text=uwhsy
  • schema: https
  • autorità(solo host name): translate.google.it
  • percorso vuoto
  • query: op=translate&sl=en&tl=it&text=uwhsy
